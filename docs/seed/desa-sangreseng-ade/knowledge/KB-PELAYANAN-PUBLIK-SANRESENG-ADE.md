# Panduan Alur Pelayanan Publik Digital (KB Tambahan)

Catatan:
- Konten ini untuk basis pengetahuan teks (KB), bukan dokumen upload.
- Informasi bersifat dummy untuk kebutuhan testing.

## 1) Kanal Pelayanan
- WA/ Webchat untuk pengajuan dan komunikasi.
- Kantor desa untuk verifikasi fisik bila diperlukan.

## 2) Tahap Layanan (Umum)
1. Pengajuan oleh warga.
2. Verifikasi awal oleh admin.
3. Permintaan perbaikan data (jika perlu).
4. Pemrosesan dan penjadwalan pengambilan.

## 3) Status & Pengambilan
- Status berubah saat berkas lengkap dan diproses.
- Pengambilan dokumen diinformasikan lewat WA/Webchat.

## 4) Tips Mempercepat Proses
- Kirim berkas yang jelas dan sesuai format.
- Gunakan nama file singkat dan konsisten.
- Pastikan nomor WA aktif agar mudah dihubungi.

## 5) Catatan Layanan Online
- Beberapa layanan tetap memerlukan kehadiran fisik untuk tanda tangan.
- Informasi detail per layanan akan muncul di form masing-masing.
