# Glosarium Istilah GovConnect (KB Tambahan)

Catatan:
- Konten ini untuk basis pengetahuan teks (KB), bukan dokumen upload.
- Informasi bersifat dummy untuk kebutuhan testing.

## Istilah Umum
- **Layanan**: Permohonan surat/administrasi.
- **Pengaduan**: Laporan kejadian/masalah warga.
- **Nomor Layanan (LAY-...)**: Identitas pengajuan layanan.
- **Nomor Pengaduan (LAP-...)**: Identitas laporan pengaduan.
- **Status Baru**: Pengajuan masuk dan menunggu verifikasi.
- **Status Proses**: Sedang dikerjakan oleh petugas/admin.
- **Status Selesai**: Pengajuan tuntas; dokumen siap/aksi selesai.
- **Status Ditolak**: Pengajuan tidak dapat diproses (data tidak valid/kurang).

## Istilah Teknis
- **Embedding**: Proses mengubah teks menjadi vektor untuk pencarian.
- **KB (Knowledge Base)**: Artikel teks yang dikelola admin.
- **Dokumen**: File yang diunggah dan diindeks sebagai sumber pengetahuan.
