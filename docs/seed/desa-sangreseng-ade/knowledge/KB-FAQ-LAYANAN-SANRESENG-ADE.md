# FAQ Penggunaan GovConnect (KB Tambahan)

Catatan:
- Konten ini untuk basis pengetahuan teks (KB), bukan dokumen upload.
- Informasi bersifat dummy untuk kebutuhan testing.

## A. Pengajuan & Perubahan Data
**Q: Saya salah pilih layanan, bagaimana?**
A: Balas pesan dengan “ubah layanan” dan sebutkan layanan yang benar.

**Q: Bagaimana memperbarui data yang sudah terkirim?**
A: Kirim pesan “ubah data” diikuti informasi terbaru.

**Q: Bolehkah satu akun mengajukan banyak layanan?**
A: Boleh, namun sebaiknya ajukan satu per satu agar proses tidak tertukar.

## B. Berkas & Format
**Q: Format file apa yang diterima?**
A: Umumnya PDF/JPG/PNG. Jika format tidak sesuai, admin akan meminta ulang.

**Q: Bagaimana jika file buram?**
A: Kirim ulang foto/scan dengan pencahayaan cukup dan posisi dokumen rata.

## C. Status & Notifikasi
**Q: Bagaimana cek status layanan/pengaduan?**
A: Kirim “cek status” diikuti nomor layanan atau nomor pengaduan.

**Q: Apakah saya akan dapat notifikasi?**
A: Ya, setiap perubahan status akan dikirim melalui WA/Webchat.

## D. Akses & Keamanan
**Q: Data saya aman?**
A: Data digunakan untuk pelayanan dan hanya diakses oleh admin berwenang.

**Q: Bisa pakai nomor WA orang lain?**
A: Disarankan memakai nomor sendiri agar verifikasi lebih mudah.
