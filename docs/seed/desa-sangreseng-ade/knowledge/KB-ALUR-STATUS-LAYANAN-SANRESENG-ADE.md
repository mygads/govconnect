# Alur Status Layanan & Pengaduan (KB Tambahan)

Catatan:
- Konten ini untuk basis pengetahuan teks (KB), bukan dokumen upload.
- Informasi bersifat dummy untuk kebutuhan testing.

## Status Layanan
- **Baru**: Pengajuan diterima dan menunggu verifikasi.
- **Proses**: Berkas dinyatakan lengkap dan sedang diproses.
- **Selesai**: Dokumen/layanan sudah siap.
- **Ditolak**: Pengajuan tidak dapat diproses (data tidak valid/kurang).

## Status Pengaduan
- **Baru**: Laporan masuk dan menunggu verifikasi.
- **Proses**: Laporan ditindaklanjuti petugas.
- **Selesai**: Penanganan selesai.
- **Ditolak**: Laporan tidak memenuhi kriteria.

## Komunikasi Status
- Perubahan status akan dikirim melalui WA/Webchat.
- Warga dapat meminta update dengan format “cek status <nomor>”.
