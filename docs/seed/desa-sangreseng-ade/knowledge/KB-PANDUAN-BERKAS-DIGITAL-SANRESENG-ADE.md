# Panduan Berkas Digital (KB Tambahan)

Catatan:
- Konten ini untuk basis pengetahuan teks (KB), bukan dokumen upload.
- Informasi bersifat dummy untuk kebutuhan testing.

## Format & Kualitas
- Format umum: PDF, JPG, PNG.
- Gunakan pencahayaan cukup dan dokumen rata.
- Hindari watermark atau stiker yang menutupi isi dokumen.

## Ukuran & Pengiriman
- Jika file terlalu besar, kompres sebelum dikirim.
- Kirim file satu per satu agar lebih mudah diverifikasi.

## Penamaan File
Contoh pola penamaan:
- NIK_Nama_Layanan_Tanggal
- 7371xxxxxx_KetUsaha_2026-01-26

## Privasi
- Pastikan hanya dokumen yang relevan yang dikirim.
- Hindari mengirim data sensitif yang tidak diminta.
